# Maximum-capacity
This is a shell script that shows battery maximum capacity.

## Usage
### Using Termux with root
```
wget https://raw.githubusercontent.com/NotZeetaa/Maximum-capacity/main/capacity.sh && chmod +x capacity.sh && su -c './capacity.sh'
```
